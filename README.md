## My Project :
**Baytak:**
- **Overview:**Create a website to provide the following home services: plumbing, electricity, cleaning, filling the water tank, and moving household furniture from city to city.
- **Features:**
- Create user profile and update.
- User authentication with roles.
- User requests a new service.
- User confirms the execution of the request.
- User can view previous orders.
- User ratings and reviews
-Search and categorization of services.
- Admin can view all user requests.
